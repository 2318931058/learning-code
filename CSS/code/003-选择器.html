标签选择器：使用标签名作为选择器，即选择同名标签设置相同的样式，如p标签，div标签等，缺点是无法差异化同名标签的显示效果
类选择器：查找对应类名的标签，差异化设置标签的显示效果
id选择器一般配合js使用，且同一个id选择器在一个页面只能使用一次
后代选择器：选中某元素的后代元素
子代选择器：选中某元素的子代元素(最近的子代元素)
并集选择器：选中多组标签设置相同的样式。 使用方式为选择器1, 选择器2,···,选择器n{属性名:属性值}
交集选择器：选中同时满足多个条件的元素(标签选择器须在最前面)

伪类选择器：伪类表示元素状态，选中元素某个状态设置样式。如：鼠标悬停 选择器:hover等
